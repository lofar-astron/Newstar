Program DWARF: private keyword MESSAGEDEVICE

Prompt: TERMINAL, PRINTER Device(s) for messages
Default: TERMINAL.
Expected input: Character(8), 2 values.

Defines where messages will be shown: On the terminal and/or in a file

to be spooled automatically to the line printer

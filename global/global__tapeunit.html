Program GLOBAL: private keyword TAPEUNIT

Prompt: MTA0, MTB0
Expected input: Character(4).

Defines the tapeunit that have to be used

MTA0 is the fast selfloading tapeunit MTB0 is the slower tapeunit (the left one)

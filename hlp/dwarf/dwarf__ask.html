Program DWARF: private keyword ASK

Prompt: YES, NO
Default: NO.
Expected input: Character(4).

ASK=YES directs DWARF programs to always prompt for parameters, even if

an external default has been defined (through SPECIFY or otherwise). This setting can be overridden by use of the /NOASK qualifier with the EXECUTE command or with run-time parameter input.

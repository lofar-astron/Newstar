Program DWARF: private keyword EXTENDSIZE

Prompt: Default extension size in blocks for DWARF data files
Default: 64.
Expected input: Integer number; min.value: 1.000000; max.value: 512.000000.

Defines the minimum extension size to be applied by DWARF I/O routines.

An actual extension will be the maximum of this parameter and the extension requested by the program.

Program DWARF: private keyword SAVELAST

Prompt: YES, NO
Default: NO.
Expected input: Character(4).

SAVELAST=YES directs DWARF programs to save the last value typed in

during program execution as an external default for later program runs. The setting of this paramater can be overriden by use of the /[NO]SAVELAST qualifier with the EXECUTE command or with parameter input.

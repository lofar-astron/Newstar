Program GLOBAL: private keyword CHANNEL

Prompt: <DeAnza Imagechannel Nr>
Expected input: I; min.value: 0.000000; max.value: 2.000000.

Defines the image-channel on the DeAnza image-processor

Channel 0 is used as overlay-channel and should (unless necessary) not be used

Program NCOPY: private keyword OPTION

Prompt: COPY, SHORTCOPY, OVERVIEW, QUIT
Expected input: Character(12), 1 values.

Actions:

COPY Copy sectors from one SCN file to another. SHORTCOPY As copy, but only scans selected by scan number; this option allows you to cut out the invalid trailing scan from a mosaic sector

OVERWIEW Display and log an overview of all sectors in a SCN file QUIT Exit from NCOPY

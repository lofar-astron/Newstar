Program NPLOT: private keyword OPTION

Prompt: Type of data to plot| MAP; DATA, RESID, MODEL; TELESC, INTERFM, IFDATA;|SPECIAL; QUIT
Default: QUIT.
Expected input: Character(24).

Specify type of data to plot:

.WMP-file data:

MAP image(s) from .WMP file

.SCN-file visibilities: DATA observed visibilities MODEL model visibilities RESIDUAL visibility residuals (after correction of all known errors and division by the visibilities of a source model (yet to be specified) (sets I=1, QUV=0)

.SCN-file correction parameters:

TELESCOPE telescope phase/gain corrections INTERFEROMETER interferometer phase/gain corrections (i.e. all corrections combined per interferometer) ! I rechecked this 960523 - JPH IFDATA IF-data: total powers, system temperatures etc.

Plotting versus sidereal time i.s.o. hour angle:

SPECIAL will prompt for a special mode

QUIT terminate NPLOT

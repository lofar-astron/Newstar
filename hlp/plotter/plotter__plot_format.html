Description of general keyword PLOT_FORMAT

Prompt: A<n>-format of plot 0, 1, 2, 3, 4
Default: 4 /ASK.
Expected input: Character(1).

Select format for (Encapsulated) PostScript plots: 0 = A0 1 = A1 2 = A2 3 = A3 4 = A4

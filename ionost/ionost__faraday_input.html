Program IONOST: private keyword FARADAY_INPUT

Prompt: YES, NO (Is Faraday rotation input required?)
Default: NO /NOASK.
Expected input: Character(3).

Faraday input flag: = NO no input data for Faraday rotation are used = YES input data for Faraday rotation are used

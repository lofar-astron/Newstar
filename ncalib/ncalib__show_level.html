Program NCALIB: private keyword SHOW_LEVEL

Prompt: Levels of terminal (1st value)| and log (2nd value) output
Default: 1, 1 /ASK.
Expected input: Integer number, 2 values; min.value: 0.000000; max.value: 4.000000.

Specify the level of the type and print output you want:

0 none 1 sector numbers, errors and summary of results only (the summary in the log file includes phase constraints and solution details which are not shown on the terminal) 2 detailed report (several pages!) per integration interval (parameter HA_INTEGRATION) - intended primarily for debugging purposes

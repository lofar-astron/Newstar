Program NCALIB: private keyword WEIGHT_MIN

Prompt: Relative minimum weight accepted
Default: 0.01.
Expected input: Real number; min.value: 0.000000; max.value: 1.000000.

Specify the minimum relative weight of a data point that is still acceptable.

The weight is relative to the maximum weight in the same scan, and in most cases can be seen as the minimum data amplitude accepted as fraction of the maximum in the scan.

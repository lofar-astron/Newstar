Program NCLEAN: private keyword CLEAN_LIMIT

Prompt: Limit in fraction of map maximum
Default: .1 /ASK.
Expected input: Real number; min.value: 0.000010; max.value: 1.000000.

Specify the level to which to clean in fraction of the initial map maximum.

Program NCLEAN: private keyword DATA_FACTOR

Prompt: Data multiplication factor
Default: 1. /ASK.
Expected input: Real number.

Specify the factor by which to multiply the input map-data.

This option is only relevant for DATA clean. The first input-map and all maps created by DATA clean will be multiplied by this factor.

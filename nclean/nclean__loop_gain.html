Program NCLEAN: private keyword LOOP_GAIN

Prompt: Clean loop-gain factor
Default: .4 /ASK.
Expected input: Real number; min.value: 0.010000; max.value: 1.000000.

For each component found, the antenna pattern shifted to its position is subtracted from the CLEAN window. To prevent overshoots, it is customary to subtract not the complete component but only a fraction of it. The magnitude of this fraction is defined here.

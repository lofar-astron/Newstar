Program NCLEAN: private keyword MAP_FACTOR

Prompt: Multiplication factor
Default: 1. /ASK.
Expected input: Real number.

Specify the factor by which to multiply the residual map before restoring.

Normally one should use the default value of 1; a value of 0 serves to make a map of the CLEAN components only.

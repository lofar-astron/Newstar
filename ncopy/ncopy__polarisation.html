Program NCOPY: private keyword POLARISATION

Prompt: XYX, XY, X; [YX] Select polarisations
Default: XYX.
Expected input: Character(4).

Select polarisations to be copied: XYX: XX, XY, YX, YY XY: XX, YY only X: XX only

YX: for special applications only: overwrite XX with XY, YY with YX, then output as if XY

Description of keywords for program NFLAG

  • OPTION NFLAG main branch FLAG; SHOW, QUIT
  • SUB_CUBE Type of secondary data cube YES, IFR, POL, HA; NO
  • INPUT_FILE Input .FLF file name (including extension)
  • OUTPUT_FILE Output .FLF file name (including extension)
  • FLAG_OPTION Type of flagging operation| CLIPDATA, DETERM; CLEAR, MANUAL, HASCANS, FCOPY;|-FLIST; MODE; INSPECT, STATIST, QUIT
  • FLAG_MODE HYPER, SECTORS, NODE; CORR, NOCORR, UFLAG;|-DRYRUN, NODRYRUN, SHOW, NOSHOW, NOTRACE, TRACE; QUIT "Set ONE environment parameter at a time; QUIT or <CR> when done|"
  • OPS_FLIST Flag-list operation| DELETE, GET, PUT; LOAD, UNLOAD, READ, WRITE; HEADER, LIST, LCOUNT;|-INSPECT, STATIST, MODE, CLEAR, QUIT
  • OPS_MANUAL Manual-flagging operation| CLEAR, CLDAT, CLHEAD; UVDATA, HARANGE; INSPECT, STATIST, MODE, QUIT
  • OPS_FCOPY Flag-copy operation TOTEL, TOPOL, TODATA, TOHEAD;|INSPECT, STATIST, MODE, CLEAR, QUIT
  • OPS_SCANS Select scan-statistics flagging criterion | SCANS; MAXABCS; ANOISE, XAN, YAN, RNOISE, XRN, YRN;|-INSPECT, STATIST, MODE, CLEAR, QUIT
  • OPS_CLIPDATA Select data-thresholding criterion | AMPL, COS, SIN; ARESID, RRESID; QXY, UXY, VXY; DT1;|-INSPECT, STATIST, MODE, CLEAR, QUIT |
  • OPS_DETERM Select coordinate criterion | SHAD, PBAS , ELEVATION; REDUN, NONRED; INSPECT, STATIST, MODE, CLEAR, QUIT
  • OPS_STATIST "Statistics operation - (ACC, SCANS, UVDAT, <name>;|-GROUPS, EXPLAIN; INSPECT, MODE, QUIT)"
  • OPS_INSPECT "Projection (COUNT, FTYP, TEL, IFR, HA) with optional '_' plus|-range modifier (X, Y, XY, YX, MAN, CLIP, NOISE, ADD, SHAD, U3, U2, U1, OLD)|-OR navigation control (STATIST, MODE)"
  • USER_FLAGS Select flag type(s) for ALL flagging operations| NONE, [ALL]; MAN, CLIP, NOISE, ADD, SHAD, U1, U2, U3; [OLD]
  • USER_FLAG default flag type(s) NONE, ALL, MAN, CLIP, NOISE, ADD, SHAD, |U1, U2, U3; OLD
  • SELECT_FLAG Flag type(s) to use in data selection| NONE, ALL, MAN, CLIP, NOISE, ADD, SHAD , U1, U2, U3; [OLD]
  • PUT_RANGE Ranges for expansion volume around PUT points:|-chan.HA.ifr.pol
  • PUT_EXPAND_CH PUT expansion half-width in channel numbers
  • HA hour angle range to process (deg)
  • LIMIT Flagging threshold
  • CLIP_LIMIT Clip threshold (W.U.)
  • CLIP_LIMITS Clip limits
  • DT1_LIMIT Discontinuity threshold (W.U. per 10 sec)
  • LIMITS flagging limits
  • PBAS_LIMITS Projected-baseline range to be flagged (m)
  • SHADOW_DIAM Telescope shadowing diameter (metres)
  • TOH_LIMIT Maximum number of data flags to tolerate per scan
  • DRY_SCANS Number of scans to use in trial runs
  • ELEVATION_LIMIT Elevation lower limit (deg)

Program NGCALC: private keyword DATA_ACTION

Prompt: data representation SHOW, AMPLITUDE, PHASE, QUIT
Expected input: Character(24).

Specify action to perform:

SHOW show detailed cut data

AMPL show amplitude of cut data PHASE show phase of cut data

QUIT quit data part

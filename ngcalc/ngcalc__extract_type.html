Program NGCALC: private keyword EXTRACT_TYPE

Prompt: TCOR, ICOR, IFDATA, DATA, MODEL, WEIGHT, QUIT data type
Default: QUIT.
Expected input: Character(24).

Specify the type of data to extract:

DATA Observed visibilities WEIGHT Visibility weights MODEL Model visibilities

TCOR Telescope gain/phase corrections ICOR Interferometer gain/phase corrections IFDATA 'IF' data: gain/system-temperature parameters from the WSRT on-line system; you will be asked for details later.

QUIT quit EXTRACT action

Program NGCALC: private keyword NGF_NODE

Prompt: Input[+output] 'node' name
Expected input: Character(80).

This is the file from which input cuts will be read.

Except in COPY operations, it is also the file to which new cuts will be written.

Enter ** (or e.g. /user0/mydata/**) to get a list of files, then enter #<n> to select the <n>-th file from such that list.

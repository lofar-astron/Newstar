Program NGCALC: private keyword OUTPUT_NGF_NODE

Prompt: Output 'node' name
Expected input: Character(80).

This is the file to which cuts will be copied.

The source and output files MUST be different.

Enter ** (or e.g. /user0/mydata/**) to get a list of files, enter #nn to select the nn-th file from such a list.

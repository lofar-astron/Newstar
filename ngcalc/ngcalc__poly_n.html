Program NGCALC: private keyword POLY_N

Prompt: number of coefficients to fit
Expected input: Integer number; min.value: 0.000000; max.value: 10.000000.

Specify number of coefficients of the polynomial to be fitted to the data.

For CALC=POLY, this is 1 + the polynomial's degree.

For CALC=DPOLY it is the number of non-zero coefficients. You will be prompted later for there orders.

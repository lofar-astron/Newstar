Program NGIDS: private keyword CLIP_LEVEL

Prompt: Clip limit for flagging
Expected input: Real number, 1 values.

All visibilities that exceed the limit in the quantity displayed (cf. parameter DATA_TYPE) will be flagged. Select a suitable level by consulting the display. The colour bar to the left is annotated with values and you the pixel value pointed at by the cursor is shown in the upper left corner. ! {\em see parameter \textref{DATA_TYPE}{.data.type} }

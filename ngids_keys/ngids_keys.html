Description of keywords for program NGIDS

  • OPTION action MAP, DATA; DOFLAG, CLIPFLAG, FLAG; UNLOAD, WRITE, CLEAR;| -GCLEAR, NOFLAG, QUIT; POINT
  • MAP_COMPRESS display-size compression factor
  • MAP_RANGE Data-value range for diaplay
  • PLOT_TYPE IFRS, BASE, CHAN
  • DATA_TYPE AMPLITUDE, PHASE, COSINE, SINE Select visibility component to display
  • MAP_SEQUENCES Display-plane range to delete
  • BLANK_FLAGS Blank flagged data-points (Yes/No)?
  • OUTPUT_FILE (output filename)
  • ALL_CHAN Propagate flag changes to all channels (Yes/No)?
  • ALL_POLS Propagate flag changes to all polarisations (Yes/No)?
  • USER_FLAG NONE, ALL; MAN, OLD, CLIP, NOISE, ADD, SHAD, U1, U2, U3 Flags to use
  • AREA Display area (l, m, dl, dm)
  • NEXT YES, NO, ALL Load next picture
  • CLIP_LEVEL Clip limit for flagging

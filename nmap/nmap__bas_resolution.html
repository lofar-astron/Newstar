Program NMAP: private keyword BAS_RESOLUTION

Prompt: Baseline averaging interval (m)
Expected input: Real number; min.value: 9.000000; max.value: 300.000000.

Specify the width in metres of baseline over which visibilities will be averaged.

The minimum value is 9, representing the smallest baseline increment ever present in practice in a (set of) WSRT observation(s). The maximum is (arbitrarily) fixed at 300.

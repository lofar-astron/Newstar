Program NMAP: private keyword CLIP_AREA

Prompt: UV-radius range for clipping (m)
Default: 0., 100000..
Expected input: Real number, 2 values; min.value: 0.000000.

Specify the (circular) UV-plane radii (in metres) between which you want to clip the data. The default is to clip everywhere.

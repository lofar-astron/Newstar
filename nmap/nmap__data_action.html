Program NMAP: private keyword DATA_ACTION

Prompt: Action to perform on the data SHOW; NOISE, OFFSET; QUIT
Expected input: Character(8).

Specify action to perform:

SHOW show detailed map data

NOISE calculate noise OFFSET calculate noise and offset

QUIT quit data part

Program NMAP: private keyword DATA_TYPE

Prompt: NORMAL, COS, SIN, AMPL, PHASE Data transformation for display
Default: NORMAL.
Expected input: Character(8).

Specify how to transform the complex input visibilities. NORMAL is the default; the others are for special experiments and diagnostics only.

NORMAL Complex value

COS Real part SIN Imaginary part

AMPL Amplitude PHASE Phase

Program NMAP: private keyword FIELD_SIZE

Prompt: Fieldsize l, m (deg)
Expected input: Real number, 2 values; min.value: 0.001000; max.value: 180.000000.

Specify the l and m field size of the map to be transformed. The default will produce a map with a resolution of about 3.5 grid intervals per synthesized-beam half-width.

If you give a NULL answer (two double quotes), you will be prompted for the grid steps.

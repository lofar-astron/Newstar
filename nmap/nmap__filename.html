Program NMAP: private keyword FILENAME

Prompt: Name for output disk file
Expected input: Character(80).

Specify the file name (without an extension) to be used in creating a pseudo-tape output file name (e.g. FITS write).

Specify a full filename otherwise (e.g. LOAD/UNLOAD in FIDDLE).

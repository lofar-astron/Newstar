Program NMAP: private keyword LM_CENTRE

Prompt: Output map centre
Expected input: Double precision number, 2 values.

Specify the centre of the output map in the l and m direction; in arcsec with respect to the mosaic reference position.

If you specify an *, you will prompted for RA, DEC position

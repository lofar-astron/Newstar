Program NMAP: private keyword MAP_FACTORS

Prompt: Input-map multipliers
Expected input: Real number, 2 values.

Specify the factors by which the input maps have to be multiplied.

You may specify up to 8 factors which will be used as multipliers in the weighted summation the input maps you selected. If there are more maps to be summed than factors specified, the factors will be cyclically re-used.

Program NMAP: private keyword MAP_LEVEL

Prompt: Polarisation threshold (W.U)
Expected input: Real number; min.value: 0.000000.

Specify the minimum level in Wetsrebork Units that is still to be considered valid linear polarisation. Polarisation levels below this threshold will be set to zero in the output map(s).

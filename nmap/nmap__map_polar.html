Program NMAP: private keyword MAP_POLAR

Prompt: Select 1 to 4 output-map polarisation(s)| XX, XY, YX, YY; I, L, Q, U, V; XXI, XYI, YXI, YYI; II, LI, QI, UI, VI
Default: XX.
Expected input: Character(4), 4 values.

Specify up to four polarisations for the maps to make:

XX XX only XY XY only YX YX only YY YY only

I Stokes I L 'line' Stokes I: Incomplete input data (e.g. no valid XX or YY) will be filled in aassuming that the field is unpolarised (Q=U=V=0) Q Stokes Q U Stokes U V Stokes V

Each of the above may be suffixed with 'I' to indicate that visibilities must be pre-multiplied with sqrt(-1)

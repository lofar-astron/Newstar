Program NMAP: private keyword OUTPUT_LABEL

Prompt: Output label
Expected input: Integer number.

Specify the first output tape label. If this label already exists, it and all the subsequent labels will be overwritten.

Specify * or 0 to write the new label behind all existing ones.

Program NMAP: private keyword REF_COORD

Prompt: Map reference coordinates: RA, DEC (decimal deg)
Expected input: Double precision number, 2 values.

Specify (in decimal degrees) the RA and DEC of the reference coordinates to use in producing the map. The coordinate system is B1950/J2000 or apparent as defined by your value for parameter MAP_COORD.

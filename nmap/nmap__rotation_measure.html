Program NMAP: private keyword ROTATION_MEASURE

Prompt: Rotation measure
Expected input: Real number, 512 values.

Specify one (or more) rotation measure(s) (RM). Pairs of input maps will be phase-rotated to the the frequency of the first input map and averaged:

THETAn = 2 * RM * ( (c/FRQn)**2 - (c/FRQ1)**2 )

Qout = SUM( Wn * ( cos(THETAn)*Qn - sin(THETAn)*Un ) ) / SUM( Wn ) Uout = SUM( Wn * ( sin(THETAn)*Qn + cos(THETAn)*Un ) ) / SUM( Wn )

A pair of Qout/Uout will be produced for each RM given. E.g. to get average Q/U maps for rotation measures from 0 to 5 with intervals of 0.5, specify ROTATION_MEASURE=0 TO 5 BY 0.5.

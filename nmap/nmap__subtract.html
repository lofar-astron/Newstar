Program NMAP: private keyword SUBTRACT

Prompt: Model subtraction (Yes/No) ?
Expected input: Logical.

Reply YES if you want to subtract a source model. You will then be prompted to provide details on the model you want to subtract. ! {\em see the \textref{NMODEL HANDLE}{nmodel_public_intfc} interface }

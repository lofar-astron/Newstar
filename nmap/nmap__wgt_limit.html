Program NMAP: private keyword WGT_LIMIT

Prompt: Relative weight limit
Expected input: Real number; min.value: 0.000000; max.value: 1.000000.

Specifies the relative weight as compared to the expected maximum weight of data points combined on one line, below which no output will be generated

Program NMODEL: private keyword ion of model file if necessary

Prompt: Finding mode POS, ABS, MANUAL; QUIT
Expected input: Character(24).

Specify how to select candidate sources:

POS Select sources with highest positive brightness ABS Select sources with highest absolute brightness MANUAL Select sources manually with the cursor on a GIDS screen. Sources are selected by pushing MB1; MB3 will finish; MB2 will not calculate position, but put grid point as clean source

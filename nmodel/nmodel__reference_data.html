Program NMODEL: private keyword REFERENCE_DATA

Prompt: RA, DEC, Freq, rot., instr.
Expected input: Double precision number, 5 values.

Specify

RA, DEC (decimal degrees) precession rotation angle (degrees) spectral-index reference frequency (MHz) instrument code (0=WSRT; 1=ATCA)

for the new reference source list position.

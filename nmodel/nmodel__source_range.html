Description of general keyword SOURCE_RANGE

Prompt: Source number range
Expected input: Integer number, 2 values.

Specify the first and last source number to process. * indicates 'all'. If you want all sources from some number <N> upward, reply

'<N>-<some very large number>'. !! Does * work?

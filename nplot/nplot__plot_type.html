Program NPLOT: private keyword PLOT_TYPE

Prompt: Data representation(s) CONTOUR, HALFTONE, RULED; POLARISATION
Expected input: Character(16), 4 values.

Specify the (combination of) data representations:

CONTOUR Contour plot HALFTONE Halftone plot * Equivalent to CONTOUR, HALFTONE RULED Ruled-surface

POLARISATION Pseudo-vectors of linear polarisation. This requires two input maps, one holding polarisation strengths sqrt(Q*Q+U*U) and the other position angles atan(U/Q)/2. ! {\em Such maps are prepared with ! \whichref{NMAP FIDDLE xxx}{} }

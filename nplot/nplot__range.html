Program NPLOT: private keyword RANGE

Prompt: Halftone saturation limits
Expected input: Real number, 2 values.

Specify the range of values to be covered by the full range of halftone shades.

The first value is the minimum to be represented by 'white', the second value the to be represented by 'black'.

NOTES: Values outside this range will always be white. (If you think this is a bad idea, please submit a Bug Report.) It is not possible to invert the scale by specifying a maximum<minimum.

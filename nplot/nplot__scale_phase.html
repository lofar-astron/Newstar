Program NPLOT: private keyword SCALE_PHASE

Prompt: Phase scale (W.U./mm or deg/mm)
Default: 1..
Expected input: Real number; min.value: 0.000000.

Specify the phase scale:

in Westerbork Units /mm for residuals in degrees/mm for source/model visibilities; for telescope corrections.

The default is a value that is likely to give reasonable output.

Program NPLOT: private keyword SIZE

Prompt: Plot scaling factors (horizontal, vertical)
Default: 1., 1. /ASK.
Expected input: Real number, 2 values; min.value: 0.001000; max.value: 5.000000.

At this point, the plot has been dimensioned to fit on a single plotter page or terminal screen, but will not necessarily fill it. You may blow it up in either or both dimensions with the factors you specify here.

If necessary, the blown-up plot will be distributed over more tham one page. !! Explain default size !! ''(i.e. a QMS plot of a power of 2 length will be 12.8 cm).''

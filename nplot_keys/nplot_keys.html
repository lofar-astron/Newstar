Description of keywords for program NPLOT

    • See also PLOTTER

  • OPTION Type of data to plot| MAP; DATA, RESID, MODEL; TELESC, INTERFM, IFDATA;|SPECIAL; QUIT
  • HA_MODE Special HA plot coordinates ST, IST, SEQUENCE, ISEQUENCE
  • HA_INTEGRATION Integration time (sec)
  • ANNOTATION Annotation text, up to 80 characters in double quotes
  • IF_MODE Parameter to plot TPON, TPOFF, TSYS, ISYS, GAIN; GNCAL, TSYSI, TNOISI, RGAINI
  • IFR_MODE NORMAL, INVERT, SORT; SPECTRAL, BAND Select data cross-section
  • ANGLE_WMP_SET ONE position-angle map: grp.fld.chn.pol.0.seq
  • DATA_TYPES Visibility component to plot AMPLITUDE, PHASE; AGAIN, PGAIN; COSINE, SINE; AP, CS
  • PLOTS_PER_PAGE Number of plots per page in hor. and vert. directions
  • DATA_TYPE DATA; SLOPE data types to plot
  • PLOT_TYPE Data representation(s) CONTOUR, HALFTONE, RULED; POLARISATION
  • SCALE_AMPL Magnitude scale (W.U./mm or %/mm)
  • SCALE_PHASE Phase scale (W.U./mm or deg/mm)
  • HA_SCALE HA plot scale (degree/cm)
  • POL_SCALE Polarisation pseudo-vector length scale W.U./cm
  • RULE_SCALE Ruled-surface height scale (W.U./cm)
  • AREA Area centre (l, m) and width (dl, dm) in grid units
  • SIZE Plot scaling factors (horizontal, vertical)
  • FULL_CONT Full-contour levels
  • DOT_CONT Dotted-contour levels
  • HALFTONE NONE, CONTINUE; STEP, PATTERN Halftone transfer function
  • RANGE Halftone saturation limits
  • TRANSFORM Grey-scale transfer coefficients:|- up to 5 groups of 2 limits plus 3 coefficients of quadratic|
  • POL_RANGE Polarised-flux limits (W.U.)
  • POL_TYPE POL, MAG Polarisation representation
  • RULE_RANGE Ruled-surface intensity range (W.U.)
  • COORD | NONE; LM, DEGREE, RADEC; DLM, DDEGREE, DRADEC;|-ONONE; OLM, ODEGREE, ORADEC; ODLM, ODDEGREE, ODRADEC Axis annotation style
  • COORD_TYPE TICK, DOTTED, FULL Coordinate grid style
  • PLOT_POSITIONS NO, YES, NAMES; EDIT Mark source positions
  • SOURCES Source pair for annotation
  • TEXT Annotation for source pair
  • PLOT_HEADING YES, NO Plot heading (Yes/No) ?
  • COORD_PREC Number of steps for coordinate contouring near pole

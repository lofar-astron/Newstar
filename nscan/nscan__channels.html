Program NSCAN: private keyword CHANNELS

Prompt: frequency channel number(s)
Expected input: Integer number, 512 values.

Select the frequency channels/bands to be included. NOTE that the WSRT continuum channel (i.e. the sum of all channels) always has number 0

'*' selects all channels

'<begin> TO <end>> BY <step>' selects channels <begin>, <begin>+<step>, ... up to <end>. The 'BY <step>' part may be omitted. BEWARE: The notation <begin>-<end>:<step> will be interpreted as an expression and give incorrect results

Each of the selected channels will be stored in the .SCN file as a separate 'channel' (chn) under the current 'group', 'observation' and 'field' grp.obs.fld. (Remember that .SCN file 'sectors' are identified by indices grp.obs.fld.chn.seq).

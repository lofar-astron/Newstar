Program NSCAN: private keyword IAT_UTC

Prompt: MJD, leap seconds, ...
Default: 48257, 26, 900000, 27, 900000, 27, 900000, 27, 900000, 27, 900000, 27, 900000, 27, 900000, 27, 900000, 27, 900000, 27 /NOASK.
Expected input: Real number, 20 values.

Specify the IAT-UTC values as pair(s) of numbers:

MJD at which the leap seconds occur The total number of leap seconds as from that date.

The first value is for 1 Jan 1991. Values before that are known by the program.

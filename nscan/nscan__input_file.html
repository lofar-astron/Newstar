Program NSCAN: private keyword INPUT_FILE

Prompt: input filename
Expected input: Character(80).

Specify the input disk-file name.

For the NSCAN LOAD-from-disk option, or for the NSCAN PFITS option, don't specify a file extension: it will be made by the program on the basis of the tape label number.

In the case of other NSCAN options, give the full file name.

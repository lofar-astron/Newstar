Program NSCAN: private keyword OUTPUT_FILE

Prompt: output filename
Expected input: Character(80).

Specify the full name for the output disk file.

For the NSCAN DUMP option, don't specify a file extension: it will be made by the program on the basis of the label number.

For the other NSCAN options, specify a full file-name.

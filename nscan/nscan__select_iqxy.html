Program NSCAN: private keyword SELECT_IQXY

Prompt: output polarisations (ONE 'value') | I, Q, U, V, IQ, UV, IQUV, X, XXY, YYX, Y, XY, YX, XYX
Expected input: Character(4).

Specify the polarisation(s) to be written:

I: I only Q: Q only U: U only V: V only

IQ: I and Q UV: U and V IQUV: I, Q, U and V

X: XX only XXY: XY only YYX: YX only Y: YY only

XY: XX and YY YX: XY and YX XYX: all four: XX, XY, YX, YY

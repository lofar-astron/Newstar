Program NSCAN: private keyword WERR_HA

Prompt: Hour-angle correction (deg)
Expected input: Double precision number in DEG, RAD, CIR, HMS.

Specify the angle to be added to all hour angles in the .SCN file

NOTE: This is a special feature to be used only to correct errors made in the on-line observation.

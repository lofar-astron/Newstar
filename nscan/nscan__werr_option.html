Program NSCAN: private keyword WERR_OPTION

Prompt: WERR action WE0, WE1; QUIT
Expected input: Character(24).

Specify correction to be performed:

WE0: correct Hour angles for tape error in splitted mosaic tapes before online version 62 WE1: correct all Hour angles with a constant offset QUIT: no more

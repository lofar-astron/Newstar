Description of general keyword SELECT_XYX

Prompt: XYX, XY, Y, X, YX, YYX, XXY polarisations
Expected input: Character(4).

Select the polarisation(s) to be used:

XYX all four combinations (XX, YX, YX, YY) XY XX and YY X XX only Y YY only YX XY and YX YYX YX XXY XY

Program DWARF: private keyword IOBUFSIZE

Prompt: Default I/O buffer size in bytes
Default: 32768.
Expected input: Integer number; min.value: 2048.000000; max.value: 32768.000000.

This is the default size of I/O buffers to be allocated by the DWARF

bulk I/O routines. This parameter is intended primarily for adapting DWARF to host systems with limited physical memory.

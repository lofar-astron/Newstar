Program DWARF: private keyword TEST

Prompt: YES, NO Set DWARF test mode
Default: NO.
Expected input: Character(4).

In testing mode:

Parameters with the TEST attribute will be prompted for. The debugger will be automatically invoked at the instant an error is reported. (Note that this may already happen during the execution of the remainder of the SPECIFY DWARF command. In that case, just type "GO" in reply to debugger prompts, and "EXIT" when the debugger reports program completion.)

Description of general keyword FLF_NODE

Prompt: Input/output .FLF file name
Expected input: Character(80).

Specify the file name (no extension).

You may enter

[<directory>/]**

to get a list of .SCN files in your current or another directory; then enter

#<n>

to select the <n>'th file from that list.

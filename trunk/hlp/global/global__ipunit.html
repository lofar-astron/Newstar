Program GLOBAL: private keyword IPUNIT

Prompt: <DeAnza-Unitnr>
Default: 0 /NOASK.
Expected input: I; min.value: 0.000000; max.value: 0.000000.

Defines the unitnr of the DeAnza image-processor

0 is the DeAnza in the image-room

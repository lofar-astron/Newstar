Program GLOBAL: private keyword LUTNR

Prompt: <Colour Lookup-Table Nr>
Default: 0 /NOASK.
Expected input: I; min.value: 0.000000; max.value: 3.000000.

Defines the nr of the colour lookup-table, which drives the colours

on the screen. 0 should normally be used

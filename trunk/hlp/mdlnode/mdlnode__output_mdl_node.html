Description of general keyword OUTPUT_MDL_NODE

Prompt: output .MDL file name
Expected input: Character(80).

Specify the output .MDL file name.

Enter ** (or e.g. /user0/mydata/**) to get a list of .MDL files in your current (or the indicated) directory; then enter #<n> to select the <n>th file from such a list.

Program NCLEAN: private keyword CMEMORY_USE

Prompt: UV-clean memory size
Default: 150000 /NOASK.
Expected input: Integer number; min.value: 32000.000000; max.value: 128000000.000000.

Specify the work memory size in bytes for the UV Clean option, to be allowed in defining the beam patch and to be used in executing the Fourier transforms.

The default shown is normally adequate; a larger value may speed up the execution of major cycles in UV Clean. !. !NCLEAN will not accept a value in excess of 32000000 (32 MB).

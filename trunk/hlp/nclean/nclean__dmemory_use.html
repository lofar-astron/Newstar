Program NCLEAN: private keyword DMEMORY_USE

Prompt: Work memory size
Default: 300000.
Expected input: Integer number; min.value: 32000.000000; max.value: 32000000.000000.

The memory workspace in bytes needed is 12 times the size of the rectangle enclosing all selected areas in the map plane. NCLEAN normally allocates up to 300 KB for this purpose. To satisfy the present need, you must either accept the value suggested here or specify a smaller set of areas, or both. !. !NCLEAN will not accept a value in excess of 32000000 (32 MB).

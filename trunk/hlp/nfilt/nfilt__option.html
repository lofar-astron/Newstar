Program NFILT: private keyword OPTION

Prompt: CONTINUUM, QUIT Action
Expected input: Character(24).

Specify the action to be performed by the program NFILT:

CONTINUUM make a UV-based estimate of the continuum in line data. The algoritm used fits a polynomial to the sine and cosine components of the residual corrected data, and stores it as additive interferometer data. Second order effects could necessitate an iteration for the cross-polarised channels to properly cater for polarisation correctioons and Faraday rotation. Although the program will handle any combination of input sets, fastest operation is attained if loops are used to select the fields.

QUIT: leave the program NSCAN

Description of general keyword MEMORY_USE

Prompt: Work memory size
Default: 200000 /NOASK.
Expected input: Integer number; min.value: 20000.000000; max.value: 4000000.000000.

Specify an approximate value for the work memory size to be used in the transform (bytes).

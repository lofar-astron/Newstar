Description of general keyword OUTPUT_WMP_NODE

Prompt: Output .WMP file name
Expected input: Character(80).

Specify the file name (no extension).

You may enter '[<directory>/]**' to get a list of .WMP files in your current or another directory; then enter '#<n>' to select the <n>th file from that list.
